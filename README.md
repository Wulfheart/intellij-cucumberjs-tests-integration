# Cucumber.js Test Runner

![Build](https://github.com/Wulfheart/intellij-cucumberjs-tests-integration/workflows/Build/badge.svg)

<!-- Plugin description -->
**Cucumber.js Test Runner** adds first-class support for running Cucumber.js tests directly from your IntelliJ-based IDE.

### Features

- **Run line markers in Gherkin files** — click the gutter icon next to any scenario or feature to run it instantly
- **Custom run configuration** — configure Cucumber.js test runs with full control over working directory, environment variables, and command-line options
- **Integrated test results** — view test output directly in the IDE's Run tool window

### Requirements

- IntelliJ IDEA Ultimate, WebStorm, or another JetBrains IDE with JavaScript support
- The [Cucumber.js](https://plugins.jetbrains.com/plugin/7418-cucumber-js) plugin must be installed
<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Cucumber.js Test Runner"</kbd> >
  <kbd>Install</kbd>

- Manually:

  Download the [latest release](https://github.com/Wulfheart/intellij-cucumberjs-tests-integration/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
